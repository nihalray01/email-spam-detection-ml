# Email Spam Detection

A Machine Learning project that classifies emails/messages as Spam or Not Spam using Natural Language Processing (NLP).

## How to Run the Project

### Step 1: Clone the Repository

```bash
git clone https://github.com/nihalray01/email-spam-detection-ml.git
cd email-spam-detection-ml
```

### Step 2: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Run the Application

```bash
streamlit run app.py
```

### Step 4: Open Browser

Open:

```text
http://localhost:8501
```
